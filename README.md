#TechLink
A platform to connect all IT Professionals, Students and Enthusiasts for sharing and contributing IT related things.

#Features :- 
- Idea Sharing
- Product Marketplace
- Job Vacancies and applying
- Podcasts sharing and listening
  
#Prerequisites :- 
- Flutter SDK (version 3.29.1)
- Dart (version 3.7.0)
- Android Studio
- VS Code
- Git
